1 History

1.1 1.5.1

The changes introduced in fswatch 1.5.1 are the following:

  • fswatch-run scripts have been removed.
  • As a consequence of the fswatch-run removal, dependency on at least one supported shell (Zsh and Bash) have been removed from the currently supported ports:
    • Macports.
    • Homebrew.
    • FreeBSD.

1.2 1.5.0

The changes introduced in fswatch 1.5.0 are the following:

  • Added the --event option to allow filtering by event type.
